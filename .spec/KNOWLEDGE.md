# KNOWLEDGE - ドメイン知識・調査結果

## プロジェクト・テーマ: 「蒼銀の断罪」
- ユーザーに「予測不能な出会い」と「能動的な情報の取捨選択」を促す。
- 「情報の深淵（アビス）」から必要なものを抽出し、不要なものを「断罪（パージ）」する体験を提供。

## 技術的な知見
- **位置ベース・インタラクション**: 
  - `VideoCard` 内に複数の `TouchableOpacity` をレイヤーとして重ねることで、視覚的なボタンを増やさずに多機能化（リフレッシュ、再生、履歴）。
- **ストック・バッファ・システム**: 
  - APIから大量の動画（100件〜）を取得し、`stock` ステートに保持。行単位でのリフレッシュ時にそこから4件ずつ「装填」することで、ネットワーク待機時間をなくす。
- **LayoutAnimation**: 
  - React Native 標準の `LayoutAnimation` を使うことで、複雑な動画の入れ替えやブロック時の詰め込みを滑らかに演出。
- **AsyncStorageによるブロックリスト管理**:
  - ユーザーが「断罪」した動画IDを永続化し、起動時にフィルタリング。
- **検索モードの配色と状態管理**:
  - `isSearching` ステートに応じた条件付きスタイルを適用。背景を紺碧（`#0d1117`）に変更し、視覚的なモード切り替えを実現。
  - `searchRows` ステートで検索結果を保持し、メインフィードとのシームレスな行き来を可能に。
- **Jujutsu (jj) による復旧の教訓**:
  - 作業ディレクトリが空に見える場合は `jj restore --from <ID>` で物理ファイルを強制復旧できる。
  - 依存関係（`node_modules`）が壊れた場合は、一旦削除して `npm install` するのが最も確実。

## 決定事項と理由
- **バックエンド: GAS/スプレッドシートへの完全移行**: 
  - Firebaseに頼らず、軽量で管理しやすいスプレッドシートをDBとして活用。
- **カテゴリ混合スキャン**: 
  - 多様なカテゴリ（アニメ、ゲーム、科学等）から少なめに取得し、ランダムに混ぜることで「アビス」感を演出。
- **検索結果の「復帰ボタン」の導入**: 
  - 探索の自由度を高めるため、メイン画面から直前の検索結果へワンタップで戻れる仕組み（`resumeSearch`）を採用。
- **プロジェクト・リポジトリの隔離**: 
  - 他のプロジェクトへの誤プッシュを防ぐため、リモート設定を `tube-mobile-spreadsheet` に一本化。
